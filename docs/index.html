HoloBio

A comprehensive Python library for Digital Holographic Microscopy (DHM)

View on GitHub Get Started

Key Features

Real-Time Analysis

Live acquisition and processing of holograms at video frame rates directly from your camera.

Versatile Optics

Compatible with lens-based and lensless systems, off-axis, in-line, and telecentric setups.

Bio-Analysis Tools

Advanced tools for particle tracking, refractive index estimation, and morphological analysis.

Installation

pip install holoBio

Or install from source for development:

git clone https://github.com/SOPHIA-Research-Lab/HoloBio.git
cd HoloBio
pip install .
