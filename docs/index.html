HoloBio

HoloBio bridges advanced optics and biology with an intuitive, open-source Python interface for Digital Holographic Microscopy. Perform Real-Time and Offline reconstruction, particle tracking, and quantitative analysis for both DHM and DLHM setups—no programming expertise required.

View on GitHub Get Started

What it does

DHM Reconstruction

Support for off-axis telecentric, non-telecentric, and other configurations.

DLHM Reconstruction

Methods including Angular Spectrum, KHDI, and Multi-Angle Scanning (MAS).

Flexible Modes

Operate in Real-Time (video-rate) or Offline post-processing modes.

Tools

Tracking, phase profiles, thickness estimation, counting, area quantification, filters, and speckle toolkit.

Installation

pip install holoBio
